Welcome to the MRA tax payer registration App

Deployment

1. Navigate to the folder called Application.
2. Run the executable file called "MRAtaxpayerWS.exe". Make sure the file Newtonsoft.Json.dll is in the same folder.
3. This is a Windows environment App, so you may create a desktop link for easy access.
4. Make sure you have at least Microsoft .net Framework 4 and above installed on the machine.
5. Run the App - "MRAtaxpayerWS.exe".
